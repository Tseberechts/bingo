# Sponsors

This folder contains the logos of our sponsors.

## How it works

The system will automatically display all images from this folder in a carousel at the bottom of the bingo screen.

To add a new sponsor:
1.  Add the sponsor's logo to this folder.
2.  The logo will automatically be included in the carousel.

## Image Guidelines

For best results, please use images with the following specifications:
-   **Format**: PNG with a transparent background is preferred.
-   **Aspect Ratio**: 16:9 (e.g., 1920x1080 pixels or 1280x720 pixels).
-   **Size**: Keep the file size as small as possible to ensure fast loading times.
